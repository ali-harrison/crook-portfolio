## About

**Crook Portfolio** is a personal project aimed at showcasing my work in web development. With a focus on creativity and interactivity, this portfolio includes a variety of projects that highlight my skills in both front-end and back-end development.

## Features

- Dynamic landing page with interactive elements
- Projects section with detailed descriptions and live demos
- Responsive design, optimized for both desktop and mobile
- Integrated contact form
- Dark aesthetic with modern styling

## Tech Stack

- **Frontend**: React, React Router
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (planned)
- **Build Tool**: Vite
- **Hosting**: Vercel (planned)
